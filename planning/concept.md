# WhatToGame Plan

## Concept

**Platform:** Desktop Browser-based
**Core Idea:** Single user application that shows the latest game releases, allows user to search for games (by release date, name, platform), and a wishlist to save /remove / games (arrange order as well).

## Features

- Display wishlist.
- Save games to wishlist.
- Remove games from wishlist.
- Arrange the order of games in wishlist.
- Filter games in wishlist by platforms.
- Recommend games listing by released date.
- Filter games listing by platforms.
- Recommend games based on wishlist preference (genre, platform).

## User Flow

1. **Start Screen** → Shows "O" and "P" buttons → Click "O" then "P"
2. **Dashboard** → Manage wishlist (remove / arrange)
3. **Latest Games** → List games by latest release dates (current 2 months) → Add selected game to wishlist
4. **Recommend Games** → List games by saved games in wishlist (genre, platform) → Hidden when wishlist is empty → Add selected game to wishlist
